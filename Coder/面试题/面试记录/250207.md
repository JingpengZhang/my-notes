#### 设计图上 `0.5px` 如何实现

1. 给容器设置伪元素，绝对定位，宽高为 200%，边框为 1px，使用 `transform: scale(0.5)` 缩小伪元素到一半。
	- 优点：兼容性最好，能正常展示圆角；
2. `border-width: 0.5px`。
3. 用阴影代替边框`box-shadow: 0 0 0 .5px #000`。
4. 伪元素背景设置为线性渐变，用绝对定位，一半透明，一半有颜色。
	- 缺点：没法展示圆角
